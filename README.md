# git-tools

Simple terminal ui for work with git branches

install [`stack`](https://docs.haskellstack.org/en/stable/#how-to-install-stack)

initialise stack's project-level YAML configuration file for an existing project
```bash
$ stack init
```

for build packages
```bash
$ stack build
```

for install bins
```bash
$ stack install
```

for uninstall bins
```bash
$ stack uninstall
```

for run tests
```bash
$ stack test
```
